# wanted-pre-onboarding-frontend

이 프로젝트는 원티드 프론트엔드 인턴십 과제 제출을 위한 repository입니다.

<br />

# Description

- 언어 : typescript
- project 구성 : CRA
- 패키지 매니져 : yarn
- styled in js : emotion
- 버전 관리 : git, github

<br />

# 실행 방법

1. 프로젝트 클론

```bash
git clone https://github.com/beomseok37/wanted-pre-onboarding-frontend.git
```

2. 패키지 설치

```bash
# yarn이 없을 경우
npm install -g yarn

# 패키지 설치
yarn install
```

1. 로컬 서버 실행

```bash
yarn start
```
